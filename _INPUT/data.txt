function setupAnimations()
{
	var html = document.documentElement;
	if (html.id == "epb3sldrzr-NavDoc")
	{
		querySelector$("#epb3sldrzr-logo").classList.add("fadeInUp");
		querySelector$("#epb3sldrzr-logo").classList.add("epb3sldrzr-animated");
	}
}

_NOTES

HTML5 slide deck for online viewing
EPUB3 e-book download (offline reading)

EPUB3 Sldrzr takes a 
- simple plain-text format
- conversion to single fileset with dual purpose
- dual support: HTML5 in web browsers + EPUB3 in reading systems (iBooks, Readium, Kobo, etc.)

- Powerpoint / Keynote etc. NO
- Mobile Webkit (iOS / Safari, Android / Chrome, etc.) 
- PREFIXED_PROPERTY CSS vendor prefixes
- author basic HTML (look ma, no cloing tags!)
- animations (animate.css)
