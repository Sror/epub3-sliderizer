Animations

EPUB3-Sliderizer ships with Animate.css

Example:

_CONTENT
<img src="assets/logo.png"
class="epb3sldrzr-animated bounceInLeft my-logo">

_CSS_STYLE
img.my-logo {
-PREFIXED_PROPERTY-animation-duration: 1.5s;
-PREFIXED_PROPERTY-animation-delay: 1s;
-PREFIXED_PROPERTY-animation-iteration-count: infinite; }

Note: the epb3sldrzr-animated class controls the initial firing of animations, avoiding premature trigger during page caching in e-book readers.
