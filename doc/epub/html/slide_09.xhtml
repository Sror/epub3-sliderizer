Incremental items

  • CSS class incremental on a wrapper element
    ⇒ reveal children in sequence
  • Additional auto class ⇒ automatic reveal
  • Custom attribute data-incremental-delay
    ⇒ milliseconds offset (default: 500ms)

Example:

<ul class="auto incremental" data-incremental-delay="200">
    <li> 1 </li>
     <li> 2 </li>
     <li> 3 </li>
 </ul>
