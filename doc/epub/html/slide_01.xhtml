What is that?

EPUB3-Sliderizer creates slideshows in HTML 5 (web) and EPUB 3 (e-book) formats, based on a simple plain-text input format.

See the input file for this slidedeck. EPUB3-Sliderizer is a free and open-source tool (GitHub).

To navigate, press the • buttons below or use the keyboard: m (menu), ▴ previous / ▾ next "bullet point" (space bar also), ◂ previous / ▸ next slides.

Tip

Depending on which EPUB3 reading system you are using ("e-reader" app or device), you can navigate these slides by flipping the pages of the e-book (left/right tap or swipe).
